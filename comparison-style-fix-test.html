✅ 修复后的效果

📷 原图
VS
✨ 处理结果
修复要点：
  • ✅ width: 100% !important 强制设置容器宽度
  • ✅ justify-content: center 确保内容居中
  • ✅ max-width: 50% 限制图片区域最大宽度
  • ✅ height: 400px 固定高度确保一致性
  • ✅ !important 确保样式不被覆盖

🔧 修复代码

主要修改：
.comparison-container { display: flex; align-items: center; justify-content: center; /* 新增：确保居中 */ gap: 4%; width: 100% !important; /* 修复：强制设置宽度 */ min-height: 300px; max-height: 500px; margin: 0 auto; box-sizing: border-box; /* 新增：确保盒模型 */ } .image-section { flex: 1; display: flex; flex-direction: column; gap: 8px; min-width: 0; /* 新增：确保flex子项可以收缩 */ max-width: 50%; /* 新增：限制最大宽度 */ } .image-wrapper { position: relative; width: 100%; height: 400px; /* 修复：固定高度 */ /* ... 其他样式 ... */ } .comparison-image :deep(.van-image) { width: 100% !important; /* 修复：强制设置van-image宽度 */ height: 100% !important; } .comparison-image :deep(.van-image__img) { width: 100% !important; /* 修复：强制设置图片宽度 */ height: 100% !important; object-fit: contain; object-position: center; }
关键修复点：
  • 🔧 使用 !important 确保样式优先级
  • 🔧 设置 justify-content: center 确保对比组件居中
  • 🔧 限制 max-width: 50% 确保两个图片平分空间
  • 🔧 固定 height: 400px 确保高度一致
  • 🔧 深度选择器修复van-image内部样式
