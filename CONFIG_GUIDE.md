# ComfyUI 配置管理指南

## 🎯 概述

现在你可以在前端直接配置ComfyUI服务器地址，无需修改代码！系统已经更新为支持动态配置，可以轻松切换不同的ComfyUI服务器。

## ⚙️ 配置功能

### 📍 访问配置
1. **主页配置**: 在主页底部点击 "⚙️ 配置" 按钮
2. **调试页面**: 访问 `/debug` 页面，在API连接测试区域点击 "修改配置"

### 🔧 配置选项

#### 服务器地址 (BASE_URL)
- **当前地址**: `https://dzqgp58z0s-8188.cnb.run`
- **格式**: 完整的URL，包含协议 (http:// 或 https://)
- **示例**:
  - `https://dzqgp58z0s-8188.cnb.run`
  - `http://localhost:8188`
  - `https://your-custom-server.com:8188`

#### 客户端ID (CLIENT_ID)
- **当前ID**: `abc1373d4ad648a3a81d0587fbe5534b`
- **用途**: 标识API请求的客户端
- **格式**: 任意字符串，建议使用唯一标识符

#### 请求超时 (TIMEOUT)
- **默认值**: 300秒 (5分钟)
- **范围**: 30-600秒
- **用途**: API请求的最大等待时间

### 🚀 预设配置

#### 1. 默认配置
```javascript
{
  BASE_URL: 'https://dzqgp58z0s-8188.cnb.run',
  CLIENT_ID: 'abc1373d4ad648a3a81d0587fbe5534b',
  TIMEOUT: 300000
}
```

#### 2. 本地开发
```javascript
{
  BASE_URL: 'http://localhost:8188',
  CLIENT_ID: 'local_client_id',
  TIMEOUT: 300000
}
```

#### 3. 当前地址
使用最新提供的服务器地址。

## 🔍 连接测试

### 自动测试
配置界面提供实时连接测试功能：
1. 填写配置信息
2. 点击 "测试连接" 按钮
3. 系统会访问 `/api/system_stats` 端点验证连接
4. 显示测试结果：✅ 成功 或 ❌ 失败

### 测试端点
- **连接测试**: `GET /api/system_stats`
- **图片上传**: `POST /upload/image`
- **工作流提交**: `POST /api/prompt`

## 💾 配置存储

### 本地存储
- 配置保存在浏览器的 `localStorage` 中
- 键名: `comfyui_config`
- 格式: JSON字符串

### 配置持久化
- 配置在浏览器会话间保持
- 清除浏览器数据会重置为默认配置
- 支持导出/导入配置（可扩展功能）

## 🔄 配置更新流程

### 1. 更新服务器地址
```javascript
// 在配置界面中
BASE_URL: 'https://new-server-url.com'
```

### 2. 保存配置
点击 "保存配置" 按钮，系统会：
- 验证配置格式
- 保存到localStorage
- 更新所有API调用
- 清除旧的测试结果

### 3. 验证配置
- 在调试页面查看当前配置
- 运行连接测试验证
- 测试完整工作流

## 🛠️ 技术实现

### 动态配置系统
```javascript
// 获取当前配置
const config = getCurrentConfig()

// 更新配置
updateComfyUIConfig({
  BASE_URL: 'https://new-url.com'
})

// 重置为默认
resetToDefaultConfig()
```

### API调用适配
所有API调用现在都使用动态配置：
```javascript
// 之前：硬编码URL
fetch('https://fixed-url.com/api/prompt')

// 现在：动态配置
const config = getComfyUIConfig()
fetch(`${config.BASE_URL}/api/prompt`)
```

## 📋 使用步骤

### 快速配置新服务器
1. **打开配置**: 主页点击 "⚙️ 配置"
2. **输入地址**: 填写新的服务器URL
3. **测试连接**: 点击 "测试连接" 验证
4. **保存配置**: 确认无误后保存
5. **验证功能**: 在一键换衣页面测试功能

### 故障排除
1. **连接失败**:
   - 检查URL格式是否正确
   - 确认服务器是否运行
   - 验证网络连接

2. **配置丢失**:
   - 检查浏览器localStorage
   - 重新设置配置
   - 使用预设配置

3. **功能异常**:
   - 清除浏览器缓存
   - 重置为默认配置
   - 查看调试页面日志

## 🔐 安全注意事项

### 配置安全
- 客户端ID不应包含敏感信息
- 使用HTTPS协议保证传输安全
- 定期更新客户端ID

### 服务器验证
- 只连接可信的ComfyUI服务器
- 验证服务器证书
- 监控异常连接
